# Lung Cancer Detection Web App

This Flask-based web application allows users to upload a lung scan image and select a model (VGG16, ResNet50, Inception, Xception, or a custom CNN) to predict the presence of lung cancer.

## Features
- Upload lung scan images
- Choose from multiple ML/DL models
- Simulated predictions (for now)
- Clean UI with image preview and result
